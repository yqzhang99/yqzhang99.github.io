## 白金 (PlayStation)
- 奖杯是一个用于纪录玩家游戏成就的系统，奖杯分为4种不同类型：铜奖杯、银奖杯、金奖杯以及白金奖杯
- 奖杯的获取方式为玩家在游戏中完成特定的事件（如过关或者消灭指定数目的敌人等）或者达到游戏中的目的（如达成一定等级排名等），当玩家解锁所有其它奖杯后（不包括DLC内容新增的奖杯），就会获得白金奖杯
- 部分规模较小的游戏没有白金奖杯
